\subsection{Detailed Description}
This class implements the xorshift128+ algorithm that is a very fast, top-\/quality 64-\/bit pseudo-\/random number generator. The quality of this P\+R\+N\+G is much higher than \hyperlink{}{Random}\textquotesingle{}s, and its cycle length is 2\textsuperscript{128}~-~1, which is more than enough for any single-\/thread application. More details and algorithms can be found \href{http://xorshift.di.unimi.it/}{\tt here}. 

Instances of \hyperlink{classairhockeyjava_1_1util_1_1_random_x_s128}{Random\+X\+S128} are not thread-\/safe.

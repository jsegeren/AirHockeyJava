Detailed Description

Encapsulates a column major 4 by 4 matrix. Like the Vector3 class it allows the chaining of methods by returning a reference to itself. For example:

Matrix4 mat = new Matrix4().trn(position).mul(camera.combined);
Author

Multiplies the vectors with the given matrix. The matrix array is assumed to hold a 4x4 column major matrix as you can get from Matrix4#val. The vectors array is assumed to hold 3-component vectors. Offset specifies the offset into the array where the x-component of the first vector is located. The numVecs parameter specifies the number of vectors stored in the vectors array. The stride parameter specifies the number of floats between subsequent vectors and must be >= 3. This is the same as Vector3#mul(Matrix4) applied to multiple vectors.
